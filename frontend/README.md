# Development Set-up
## Prerequisites
* Android Phone
* Visual Studio
* Expo Go
## Instructions
1. Install `Expo Go` from Google Playstore
2. Cd to the `frontend` folder
3. Run the following command to install packages:
```
npm install
```
4. Run the following command to use the frontend:
```
npx expo start
```
5. The code may not work since .env is not part of the files. In that case, you can access the following Google Drive link for the [APK](https://drive.google.com/drive/folders/19BdkcCmJ-A0NYRhgxPIcASETtH5T5L8s?usp=sharing) of the entire application.
6. When the command from Step #4 successfully worked, it should display a QR code.
7. Open the `Expo Go` app in your Android and go to Scan QR. Afterwards, scan the QR displayed in the IDE terminal.
