# Project Details
Creation of my Special Problem titled "PawPlan:  A Mobile Application for Pet Health Management and CBC-Based Meal Generation for Adult Dogs and Cats" which incorporates mobile development, mixed-integer linear programming, and multithreading processes. It uses the following technology stack

1. React Native
2. Firebase
3. Python (PuLP for MILP)
4. FastAPI

## Structure

`frontend`: contains frontend-related code. Refer to README.md inside the folder for setup.
`solver-backend`: contains the Python code for the solver.
